PD Controller Code:

```matlab
Kp = 2;
Kd = 0.5;

s = tf('s');
C = Kp + Kd*s;

figure;
bode(C);
grid on;
title('PD Controller: Bode Plot');
```

Lead Compensator Code:

```matlab
K = 1;
tau = 0.5;
alpha = 0.1;

s = tf('s');
C = K * (tau*s + 1) / (alpha*tau*s + 1);

figure;
bode(C);
grid on;
title('Lead Compensator: Bode Plot');
```

PI Controller Code:

```matlab
Kp = 2;
Ki = 1;

s = tf('s');
C = Kp + Ki/s;

figure;
bode(C);
grid on;
title('PI Controller: Bode Plot');
```

Lag Compensator Code:

```matlab
K = 1;
tau = 0.5;      
beta = 10;      

s = tf('s');
C = K * (tau*s + 1) / (beta*tau*s + 1);

figure;
bode(C);
grid on;
title('Lag Compensator: Bode Plot');
```
